# RandomMeas.jl: Julia Interface to common random measurements

We provide scripts to reconstruct multi-copy and single-copy randomized measurements <img src="Pics/RandomMeasurements.png" alt="drawing" width="250"/>.
The library uses ITensors.jl to postprocess randomized measurements data.


## von Neumann entropy from common randomized measurements
The von Neumann entropy is obtained from common randomized measurements, via a polynomial approximation. In order to reduce statistical errors, we construct an unbiased estimator using simulated measurements on a classical computer.

+ [Original Paper](http://arxiv:)
+ [Script](vonNeumann.jl)

## Fidelities from common randomized measurements
The fidelity is obtained from common randomized measurements. We use as reference state multiple Matrix-Product-States of varying bond dimensions. This example uses the library PastaQ.jl to represent quantum states generated by random quantum circuits.

+ [Original Paper](http://arxiv:)
+ [Script](Fidelity.jl)
